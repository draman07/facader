Façader is a design tool to rapidly evaluate potential studies of building elevations. This script considers exterior cladding module and color, window size and location, and entry level height. Façader is written in Processing and uses intelligent 2D objects with collision detection to create many combinations from several base scenarios.

The contemporary future of architecture is less about explicitly creating form and more about creating systems of control for form to occur. Parametricism is new process forward that is congruent with any shape or form. The role of the Architect has shifted towards catalyst rather than pure creator.
